*   **5.8 Evaluation Metrics:** Confusion Matrix, Accuracy, Precision, Recall, F1-Score, ROC-AUC, MSE, RMSE, and R².
*   **5.9 Validation Strategies:** K-Fold Cross-Validation and train-test splitting.
*   **5.10 Hyperparameter Tuning:** Automated model optimization using GridSearchCV, RandomSearch, and Optuna.

*   **🛠️ Milestone Project 5:** Build, tune, and evaluate an End-to-End machine learning predictive pipeline on a competitive dataset.
